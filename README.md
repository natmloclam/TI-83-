# TI-83-

A hardware-based graphing calculator implemented on the Digilent Basys 3 FPGA. This project supports mathematical function visualization along with basic differentiation and integration capabilities.

## Features
- Basic calculator functions - 4 operands and exponentiation
- Graph plotting of polynomial functions
- Input limits to find definite integral of your function 
- Find the derivative at any point on your function 

## How to use

Ensure you have Vivado 2018.2 installed on your computer

1. Download the latest release 
2. Open `CedricGroup.xpr` in Vivado
3. Connect your Basys 3 FPGA to your computer
4. Navigate to the Program and Debug menu > Open Hardware Manager > Open target and select your device
5. Click `Program Device` and you are ready to go
